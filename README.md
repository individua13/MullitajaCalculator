# E-Liquid Mixing Calculator 🧪💨

A simple and intuitive web tool for e-liquid mixing calculations. Perfect for DIY vapers who need to quickly calculate base and flavoring quantities.

## 🌐 Live Demo

Try it here: [E-Liquid Mixing Calculator](https://torugurud.ee/github/ForMullitajaShop/MullitajaKalkulaator.html)

---

## ✨ Features

- **Real-time calculations** — Updates instantly as you adjust values
- **Dual volume display** — Shows both milliliters and drops
- **Responsive design** — Works on desktop and mobile devices
- **User-friendly interface** — Simple +/- buttons and manual input
- **No installation needed** — Works directly in your browser

---

## 📋 How It Works

1. **Set total volume** — Enter your desired final e-liquid volume in ml
2. **Set flavor percentage** — Choose the percentage of flavoring (0–50%)
3. **Get instant results** — The calculator automatically shows:
   - Base quantity in ml and drops
   - Flavoring quantity in ml and drops

---

## 🧮 Calculation Details

- **Drop conversion rate:** 1 ml = 33 drops (standard assumption)
- **Base volume:** Total volume − Flavor volume
- **Flavor volume:** (Flavor % / 100) × Total volume
